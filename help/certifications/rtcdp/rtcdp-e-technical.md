# Adobe 인증 여정 [!DNL Real-Time Customer Data Platform] 전문가

인증된 Adobe이 되는 방법 알아보기 [!DNL Experience Platform] Real-time Customer Data Platform(RTCDP)의 전문가.

## 최소 경험

시험에 대한 인증을 받으려면 감독자가 온라인 또는 현장 방문 중에 2개의 개별 시험을 통과해야 합니다.

다음을 수행해야 합니다.

* 에 대한 기본 지식 [!DNL Experience Platform] 및 은 UI를 통해 탐색할 수 있습니다
* 세그먼트를 만들고 대상을 빌드하는 기능(여기에 포함된 논리)
* 대상을 구성하는 기능
* 세그먼트를 대상에 연결
* 실시간으로 반응하고 반응할 수 있는 인사이트/이벤트를 빌드하는 기능
* 데이터 수집 중 DULE 설정 이해, 활성화 중 세그먼트 제한
* 채널별 옵트인 요구 사항 이해
* 사용 사례 데이터 빈도 및 볼륨에 대한 우수 사례를 사용하여 캡처 및 활성화해야 하는 데이터를 디자인/이해하는 기능
* 다양한 채널에서 필요한 데이터를 파악하는 기능
* 필요한 프로필 속성을 확인하는 기능
* 대상을 통해 세그먼트를 전송하는 빈도에 대한 지식
* 세그먼트 멤버십이 업데이트되는 빈도에 대한 정보

>[!NOTE]
>
>합격하신 후 2년간 유효한 인증입니다 최신 제품 기능을 다루므로 가능한 한 최신 버전을 사용하는 것이 좋습니다.

**시험 세부 정보:**

* 수준: 전문가(1-3년 경험)
* 점수 전달: 35/50
* 시간: 105분
* 배달: 온라인 처리(카메라 액세스 필요) 또는 테스트 센터 처리
* 언어: 영어

**시험 목표 및 범위**

섹션 1: 데이터 모델링(20%)

* 소스 데이터를 분석하여 프로필 결합을 위한 기본 및 보조 ID를 평가합니다.
* UI를 사용하여 XDM 스키마를 생성/편집하는 방법에 대한 이해를 보여줍니다.
* DULE 지침 식별

제2절: 데이터 수집(36%)

* 데이터를 수집하기 위해 형식을 지정하고 준비하는 방법을 보여줍니다.
* OOTB 커넥터를 사용하여 데이터 소스를 연결하는 방법을 보여줍니다.
* 일괄 처리 및 스트리밍을 통해 소스 데이터를 수집하는 방법을 보여줍니다.
* 데이터 전송을 모니터링하는 방법 설명
* 소스 데이터에 대한 데이터 검색을 수행하는 방법을 보여 줍니다.
* 데이터를 XDM과 일치하도록 변환하는 방법을 보여 줍니다

섹션 3: 프로필 통합(34%)

* 프로필 및 이벤트 데이터 게시 수집 유효성 검사
* ID 이름 공간 정의
* 프로필 서비스에서 ID 그래프를 사용하는 방법을 설명합니다
* 데이터 과학 작업 공간 모델링 서비스를 통해 프로필을 보강하는 방법 식별
* 세그먼트 빌더로 세그먼트를 만드는 방법과 세그먼테이션 서비스가 작동하는 방식을 보여 줍니다

섹션 4: 활성화(10%)

* 대상을 설정하는 방법 및 세그먼트 활성화가 작동하는 방식을 보여 줍니다
* Real-Time CDP을 통해 데이터 액세스 API와 데이터 내보내기에 대한 데모 및 이해

최소 자격을 충족하는지 알아보기 위해 준비 설문지를 작성하세요.
