# Adobe 인증 여정 [!DNL Experience Manager Forms] 개발자 전문가

인증을 받는 방법 알아보기 [!DNL Experience Manager Forms] 개발자 전문가.

## 최소 경험

다음을 수행해야 합니다.

* AEM Forms 1-3년 경력
* IT 업계에서 최소 5년 근무 경력
* 사용자 지정된 양식 솔루션의 개발, 구성, 테스트 및 배포를 주도하는 기능
* 양식을 구성하고 비즈니스 요구 사항을 워크플로우로 변환하고 외부 시스템과 통합할 수 있는 기능
* AEM Forms 기초 구성 요소에 대한 이해

또한 다음 기술 및 환경에 익숙해야 합니다.

* XDP 템플릿, XFA 템플릿 디자인
* PDF forms 만들기(정적/동적, 대화형 및 비대화형)
* 스키마 또는 XML 데이터 바인딩
* 양식 조각 만들기
* 적응형 양식 만들기
* 복잡한 양식 디버깅
* 양식 제출 구성 및 수행
* 적응형 양식을 위한 AEM 인스턴스 구축 및 배포
* 양식 모델 사용
* 데이터 서비스와 통합 기능
* 워크플로우 구현
* 긴 라이브 - 짧은 라이브 프로세스 및 서비스를 만들기 위한 AEM Forms Workbench(JEE)에 대한 기본 이해
* GuideBridge API를 사용하여 Javascript clientlibs 개발
* Sign v 정부 CAC를 기반으로 사용할 AEM Forms 환경 데스크탑 또는 서버를 파악합니다
* 논리를 작성하고 다른 서비스를 호출하는 Java/J2EE 및 JavaScript에 대한 기본 이해.
* 적응형 양식 미리 채우기
* OSGi, Sling 구성 요소에 대한 기본 이해 또는 지식입니다
* 문서 서비스 사용
* AEM Designer, AEM Workbench, AEM 작업 공간
* 적응형 양식 콘솔
* Java IDE 도구
* Maven
* GIT/버전 관리 도구

* XFA/XDP 디자인에서 다른 출력을 생성합니다
* Forms 또는 어셈블러 서비스를 사용하여 PDF 조작
* JDBC 서비스 구현

섹션 3: 관리 및 문제 해결(18%)

* 로그 파일 문제 해결 및 차별화
* 관리 활동 수행
* 서비스 팩 적용
* Forms에 대한 AEM Dispatcher 설정 수행
* 다른 콘솔 탐색

섹션 4: API 엔드포인트 및 통합(8%)

* Adobe Experience Cloud와 통합
* 다양한 끝점 서비스 생성 및 사용

* 대화형 커뮤니케이션을 위한 프로세스 디자인, 데모 및 구현
