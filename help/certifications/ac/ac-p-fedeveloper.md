# Adobe 인증 여정 [!DNL Commerce] 프런트엔드 개발자 전문가

인증된 Adobe Developer Professional이 되는 방법을 알아봅니다.

## 최소 경험

Adobe에 대해 0-12개월 실무 경험이 있어야 합니다 [!DNL Commerce] 프런트엔드 개발 및 기술 솔루션을 사용하여 개별 구성 요소를 제공하는 기능.

또한 다음 기술에 대한 지식이 있어야 합니다.

* Adobe에 대한 강력한 이해 [!DNL Commerce] 버전 2.4.3
* 프런트 엔드 구조 및 레이아웃에 대한 기본적인 이해
* HTML, CSS(및 전처리기), LESS, JavaScript 및 UI 구성 요소, jQuery, Underscore, Require.JS, Chnokout, PHP 템플릿, DB 기술 및 컨텐츠 관리에 대한 경험과 이해
* 플랫폼, CMS 블록, 위젯, CMS 페이지, 카테고리 페이지, 제품 페이지, 장바구니 및 체크아웃, 계정 대시보드 영역에 대해 잘 알고 있습니다
* 관리 구성에 대한 기본 이해 또는 지식
* 스타일, 레이아웃 및 JS 파일 확장 및 덮어쓰는 방법에 대한 지식
* 상위 테마/테마 계층 확장
* CLI 명령(캐시 정리, 설정:업그레이드, 인덱서, 배포 모드) 및 단위
* 캐싱 기술, 클라이언트측 브라우저, 저장소 이해(쿠키, 로컬 저장소)
* 반응형 디자인 및 미디어 쿼리 이해
* WYSIWYG/Page Builder
* 코드 패키지 이해
* 번역 구현
* 이메일 템플릿 작업 방법 이해
* XML에 대한 기본 이해
* 기본 제공 SEO, 최적화 및 성능 기술에 대한 기본 이해
* 정식 링크에 대한 기본 이해 및 JavaScript 및 CSS 병합

>[!NOTE]
>
>합격하신 후 2년간 유효한 인증입니다 최신 제품 기능을 다루므로 가능한 한 최신 버전을 사용하는 것이 좋습니다.
