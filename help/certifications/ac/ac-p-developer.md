인증된 Adobe Developer Professional이 되는 방법을 알아봅니다.

## 최소 경험

Adobe에서 일하는 것이 편해야 합니다 [!DNL Commerce] Adobe에 대한 기본적인 이해를 제공합니다 [!DNL Commerce] 관리 개발, 아키텍처, EAV/데이터베이스, 레이아웃/UI, 체크아웃 및 판매, 카탈로그

또한 다음 기술 및 환경에 익숙해야 합니다.

* Adobe [!DNL Commerce] - 버전 2.4
* PHP 이해
* MYSQL 지식
* 객체 지향 프로그래밍 지식
* 클라우드 아키텍처 지식

* 시나리오가 주어지면 컨트롤러 생성
* 모듈 구조 설명
* 인덱스 기능 설명
* 현지화 설명
* 플러그인, 기본 설정, 이벤트 옵저버 및 인터셉터 설명
* 사용자 지정 모듈 경로 설명
* URL 다시 작성 설명
* 설명 [!DNL Magento] 캐싱 시스템
* 저장소, 웹 사이트 및 저장소 보기 설명(기본 이해)

섹션 3: EAV/데이터베이스(13%)

* 시나리오가 주어지면, 속성 세트 및/또는 속성을 변경/추가/제거합니다
* 다양한 유형의 속성 설명
* 시나리오가 주어지면 DB 스키마를 사용하여 데이터베이스 테이블을 변경합니다
* 모델, 리소스 모델 및 컬렉션 설명

* 다중 출처 재고가 재고(프로그램 레벨)에 미치는 영향 설명

* 장바구니 프로모션 규칙 설명
* 시나리오가 발생하면 기본 체크아웃 수정 사항을 설명합니다

* 시나리오가 주어지면 세금 규칙, 통화, 장바구니 및/또는 체크아웃을 구성합니다
