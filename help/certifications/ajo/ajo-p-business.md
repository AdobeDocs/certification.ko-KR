### 최소 경험 {#minimum-experience}

지원자는 다음 작업을 수행할 수 있어야 합니다.

* 메시지 유효성 확인 및 증명, 시드 목록
* 미리 디자인된 템플릿/드래그 앤 드롭을 사용하여 이메일, SMS, 인앱 메시지 및 푸시 알림 만들기
* 여정 테스트 및 유효성 검사
* 여정 및 캠페인(이메일, SMS 및 푸시만 해당)을 만들고 여정에 기본 조건/제외 추가
* 프로필 속성 및 이벤트 데이터를 사용한 기본 개인화 이해
* 구독 목록 사용
* 프로필 속성 및 이벤트 검토(UI에서 프로필 로드)
* 랜딩 페이지 이해 및 사용
* 다양한 유형의 여정/캠페인을 사용하는 시점을 파악하고 [!DNL Journey Optimizer] Adobe의 &quot;캠페인&quot; 기능을 활용하여 일회성 이메일 전송을 구축할 수 있습니다.
* 데이터 소스, 이벤트 및 작업 기능 이해
* 여정 보고서를 모니터링하고 검토하여 이메일 참여/바운스/오류율/스팸 비율/도메인 문제 추적
* 기본 세그먼트 기준 작성 및 유효성 검사
* 이메일 제외 및 허용 목록 이해
* 단일 및 비즈니스 이벤트 이해
* 여정의 다양한 활동 이해 및 사용
* 기본 디버깅 - 테스트 모드 또는 보고(쿼리 서비스 필요 없음)
* 비즈니스와 관련된 여정 요구 사항 수집 및 문서화
* 여정 및 채널에 대한 모범 사례(예: 전송 모범 시간, 사용 모범 채널, 다양한 사용 사례에 대한 모범 여정 디자인 패턴, 트랜잭션 및 마케팅 메시지 사용 시기)를 파악합니다
* 전달성 개념 이해(예: 다양한 바운스, 준비)
* 빈도, 피로 규칙 이해
* 데이터 세트 미리 보기
* ID에 대한 기본 이해
* 대상자 구성, DM 채널 사용 시기 이해 - 2023년 4분기 중에 시작될 예정입니다
* 정적 및 동적 오퍼 컬렉션 및 시뮬레이션 이해(오퍼 테스트)

지원자는 또한 다음을 수행할 수 있어야 합니다.

* 대상자 작성(새 대상자 작성기 사용)
* 대상자 가져오기(CSV 대상자 업로드)
* 이메일에 대한 HTML 업로드 및 자동 변환  (새 html 변환기)
* 콘텐츠 실험 만들기
* 콘텐츠 실험의 결과 측정
* 여정, 캠페인 등에 태그 지정/필터링/검색
* 템플릿에서 조각 만들기
* 사용자 지정 작업
* Asset Essentials에서 콘텐츠 만들기
* 외부 데이터 소스
* Offer decisioning(기본 지식)
* XDM(경험 데이터 모델)에 대한 기본 이해
* 데이터 가져오기 및 내보내기 모니터링 및 검토
* 콘텐츠 템플릿 만들기
* 데이터 사용 및 레이블 지정 개념

<br>

### 시험 목표 및 범위 {#exam-objectives}

{{exam-objectives}}

**섹션 1: 캠페인 및 여정 최적화(48%)**

* 시나리오가 주어지면 고객 여정을 구축하는 방법을 확인하십시오.
* 주어진 시나리오에서 고객 여정의 유효성을 검사하는 방법을 결정합니다.
* 여정 실행 후 시나리오가 주어지면 고객 여정을 평가하는 방법을 확인하십시오
* 시나리오가 주어지면 단일 이벤트 또는 비즈니스 이벤트를 사용할 시기를 식별합니다

**섹션 2: Offer decisioning(12%)**

* 오퍼 컬렉션을 만드는 방법 식별
* offer decisioning 단계 식별
* 오퍼 구성 방법 식별
* 시나리오가 주어지면 작성 중인 콘텐츠에서 오퍼 의사 결정 활동을 사용하는 방법을 식별합니다
* 시나리오가 주어지면 규모에 따라 offer decisioning 또는 개인화를 사용할 시기를 식별합니다
* 시나리오가 주어지면 정적 오퍼와 동적 오퍼를 구별합니다

**섹션 3: 콘텐츠 작성(20%)**

* Asset Essentials 사용 방법 식별
* 시나리오가 주어지면 이메일에 개인화된 필드를 추가하는 방법을 확인하십시오
* 시나리오가 주어지면 콘텐츠 실험에 적합한 구성을 식별합니다
* 시나리오가 주어지면 콘텐츠 실험을 해석하는 방법을 식별합니다
* 조각 사용의 이점 식별
* 조각 유형 식별
* 이메일 템플릿을 만드는 방법 식별

**섹션 4: AEP의 기초(20%)**

* 시나리오가 주어지면 서로 다른 유형의 데이터를 사용할 시기를 결정합니다
* 프로필 속성 및 대상자 멤버십을 검토할 위치 식별
* 프로필에 대해 활성화된 데이터 세트와 프로필에 대해 활성화되지 않은 데이터 세트 구분
* 여정에서 지원되는 다른 유형의 ID 식별
* 시나리오가 주어지면 특정 대상 유형을 사용할 시기를 식별합니다
* 시나리오가 주어지면 순차적 대상에 사용할 올바른 논리를 식별합니다
* 시나리오가 주어지면 데이터 로드 문제를 해결하는 방법을 확인하십시오
* 시나리오가 주어지면 적절한 데이터 레이블을 식별합니다
