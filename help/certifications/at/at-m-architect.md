# 에 대한 인증 여정 [!DNL Adobe Target] 아키텍트 기본

인증된 Adobe 설계자가 되는 방법을 기본.

## 최소 경험

다음을 수행해야 합니다.

* 구현을 포함하여 3년 이상의 디지털 마케팅 최적화 경험 [!DNL Adobe Target] 활동
* 구현, 배포 및 문제 해결을 위한 최소 2년의 경험(또는 5건의 참여) [!DNL Adobe Target]
* Adobe Experience Cloud과 Target 통합 경험
* JavaScript, HTML, CSS 및 프런트 엔드 웹 개발에 대한 기본적인 이해
* 에 대한 이해/인식 [!DNL Adobe Target] API/SDK, 서버측 통합, 보고 및 관리
* 구현 지식 [!DNL Adobe Target] SPA(단일 페이지 애플리케이션) 프레임워크, 모바일 및 이메일 및 권장 사항 사용

또한 다음 기술 및 환경에 익숙해야 합니다.

기술:

* 여러 Adobe Experience Cloud 프로그램(Adobe Analytics, Adobe Campaign, Adobe Experience Manager, Adobe Experience Cloud Debugger, Target API)
* JavaScript(Ajax, JQuery 등)
* 브라우저 기반 디버깅 도구(Chrome 개발자 도구, VEC helper), Target의 응답 디버깅을 위한 JSON
* 기본 앱용 디버깅 도구(Charles Proxy, Fiddler)
* HTML, CSS, HTTP, XML 구성
* 기본 모바일 개발 IDE, iOS 개발의 기본 사항, Android 개발의 기본 사항
* Postman과 같은 API 도구
* VEC helper

환경:

* Adobe Experience Cloud
* 태그 관리자 없이 클라이언트측 구현
* Tag Management 시스템, 특히 Launch(IF)
* 서버측 환경(API 사용)
* 이메일
* 웹 및 모바일 환경
* AMS 호스팅 환경
* Adobe I/O Runtime

>[!NOTE]
>
>합격하신 후 2년간 유효한 인증입니다 최신 제품 기능을 다루므로 가능한 한 최신 버전을 사용하는 것이 좋습니다.
